The Vault Developer's Version with API

After April 14, 2022, developers can download the vault locally rather than using a CDN (Content Delivery Network) to add Vault into their application. There isn't an additional charge to download the Vault API and use locally inside your Project. NOTE: Installing vault locally inside your project might increase the size of your project when compiling it.
